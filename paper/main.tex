% or rank-compared across factor types.

\subsection{$\epsG$-invariance (dual-gate definition)}
% G(F,c) <= eps_G            => invariant (report G<0 sub-case separately)

% G(F,c) > eps_G and S <= 0  => inconclusive / probe-driven (dead zone)
